# 修复：TabBar 点击不变颜色

## 问题根因
所有页面 `onShow` 中使用 `app.getTabBar()` 获取 TabBar 实例，但 `getTabBar()` 是微信小程序**页面实例**的内置方法，app 对象上没有该方法。导致 `app.getTabBar` 为 `undefined`，`setSelected()` 从未执行，选中态不会更新。

## 修复方案
将每个 Tab 页面的 `onShow` 中的 `app.getTabBar()` 改为 `this.getTabBar()`。

### 修改前（以 home.js 为例）：
```js
onShow() {
  const app = getApp();
  if (app.getTabBar) {           // ← 永远为 false
    const tabBar = app.getTabBar();
    if (tabBar) {
      tabBar.setRole(app.globalData.userRole || 'user');
      tabBar.setSelected(0);
    }
  }
}
```

### 修改后：
```js
onShow() {
  const app = getApp();
  const tabBar = this.getTabBar();
  if (tabBar) {
    tabBar.setRole(app.globalData.userRole || 'user');
    tabBar.setSelected(0);
  }
}
```
