# 首页添加轮播图

## 改动范围
仅修改 `pages/home/` 下 3 个文件

## 布局顺序
导航 → 轮播图 → 分类标签 → 信息流

## 修改内容

### 1. home.js
- data 中添加 `banners: []` 和 `bannerCurrent: 0`
- `onLoad` 中加载轮播数据（可先用静态占位数据）
- 添加 `onBannerChange` 事件

### 2. home.wxml
- 在导航栏 `</view>` 后、分类标签前插入 `<swiper>` 轮播组件
- 轮播图高度约 300rpx，圆角卡片风格
- 分类标签的 `margin-top` 从 `statusBarHeight + 88` 改为 `statusBarHeight + 88 + bannerHeight`（或改为非 fixed 定位，用 padding-top 撑开）

### 3. home.wxss
- 添加 `.banner-wrap` 样式（固定定位下的轮播容器）
- 添加 `.banner-img` 样式
- 调整 `.category-bar` 的 margin-top

## 验证
编译后首页显示：导航 → 轮播图 → 分类标签 → 信息流
